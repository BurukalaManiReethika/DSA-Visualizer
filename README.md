# 🚀 DSA Visualizer

A Python Tkinter application that visualizes:

- Stack
- Queue
- Linked List

## Features

✅ Push & Pop Operations

✅ Enqueue & Dequeue Operations

✅ Insert & Delete Nodes

✅ Interactive GUI

## Tech Stack

- Python
- Tkinter
- Data Structures

## Run

```bash
python main.py
```
